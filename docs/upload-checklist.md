# 上传材料清单

## 推荐上传顺序

1. Roblox Studio 运行截图，也就是能看到 Coin Dash UI、金币、障碍和终点门的截图。
2. 代码文件截图，展示两个核心脚本已经创建。
3. Token 预算规划图，展示预计 Token 消耗和阶段规划。
4. GitHub 仓库 README 截图或仓库链接。

## 申请表链接栏

```text
https://github.com/xingxingeyl8/coin-dash-roblox-demo
```

## 备注

Token 消耗图建议使用“预计消耗规划”更稳妥，不建议伪造成真实平台账单。真实账单或真实控制台截图只有在你确实调用过对应平台 API 后再上传。
