# 申请表填写内容

## 04 请描述你使用 Agent 或 AI 驱动构建的具体成果

我正在构建一个面向零基础创作者的 AI 游戏原型开发 Agent 工作流，用于将简单游戏玩法快速转化为 Roblox Studio 可运行 Demo。当前已完成 Coin Dash 原型：玩家需要在 90 秒内收集 10 个金币，金币收集完成后终点门开启，玩家抵达终点即可通关。

该 Demo 包含地图自动生成、金币收集机制、倒计时、胜负判定、基础 UI、音效反馈、交互反馈和 Roblox Lua 脚本结构。项目核心痛点是：非程序背景用户很难独立完成游戏原型开发，尤其是在玩法拆解、脚本编写、UI 交互和工程整理方面缺少低门槛工具。

后续计划将该流程扩展为多 Agent 协作系统：需求分析 Agent 负责拆解玩法，代码 Agent 负责生成 Roblox Lua，测试 Agent 负责根据报错迭代修复，文档 Agent 负责整理交付说明。预计 Token 消耗主要来自多轮需求分析、代码生成、错误日志诊断、玩法迭代和教学式解释。首阶段目标是沉淀 5-10 个可复用 Roblox 原型模板，并逐步形成面向游戏创作者的 AI 原型生成工作流。
